Lenguaje de marcas y SGI

INFORMACIÓN

Lenguaje de marcas es una asignatira impartida por el profesor Jorge Sanchez en la cual nos explica principalmente sobre la creación de páginas web

CRITERIOS

  • El 70% de la nota será de la media de los exámenes
    1. Exámenes teóricos
    2. Exámenes prácticos (en ordenador)
    3. Proyectos (pueden ser individuales o por equipos
  • El 20% de la nota será de la media de los trabajos
    1. Cuestionarios
    2. Test
    3. Trabajos
  • El 10% de la nota será por la asistencia a clase

TEMAS

  1. 1º Evaluación
    • Introducción a los lenguajes de marcas
    • Creación de documentos web básicos con HTML
    • Formateo de documentos HTML con CSS
  2. 2º Evaluación
    • Maquetación de aplicaciones web con HTML y CSS
    • Dinamización y adaptación de aplicaciones web con JavaScript
    • Formatos documentales XML y JSON
  3. 3º Evaluación
    • Validación de documentos XML
    • Manipulación y consulta de datos XML y JSON con JavaScript
    • Sindicación de contenidos
    • Sistemas de Gestión Empresarial

TIEMPO IMPARTIDO

  • Horas semanales:4 horas
  • Horas totales: 122horas
    1. 1º Evaluación; 32 horas
    2. 2º Evaluación; 50 horas
    3. 3º Evaluación; 40 horas
